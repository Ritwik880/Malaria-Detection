Classifying Malarial Cells using Convolutional Neural Networks




Malaria is a mosquito-borne infectious disease that affects humans and other animals. Malaria causes symptoms that typically include fever, tiredness, vomiting, and headaches. This Web Application utilizes Tensorflow and Flask to develop and deploy a Machine Learning Web Application that can detect Malaria Cell using Image Processing.
